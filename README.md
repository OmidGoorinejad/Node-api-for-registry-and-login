# Simple Authentication API (Node.js)

A simple REST API for user registration and login built with Node.js.

## Features
- User registration
- User login
- JWT-based authentication
- Password hashing
- MariaDB database

## Technologies
- Node.js
- Express
- MariaDB
- JWT
- bcrypt

## Getting Started

### Prerequisites
- Node.js installed
- MariaDB installed
